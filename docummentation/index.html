A) Database Configure - top


Path of the file: public_html > system > config.php

	
$db_host="localhost";  >  Your HOST, default is "localhost"

$db_n="database_name"; > Your MySQL username

$db_u="username"; > Your MySQL password

$db_p="database_password"; > The name of the database

$site_url="http://www.site-url.com"; > Your site url address

$ssl_status=false; > Your site's SSL status, true or false

Config settings are set automatically with quick setup. Set your SSL settings or you can use it to change the site URL.

B) Cronjob - top

You must define cronjobs settings for system properties to work properly. Market Capitalizion, New cryptocurrency, Current currency data, Created alerts is working with cronjob.

Cronjob - Market Cap.

wget -O /dev/null http://www.site-url.com/system/cronjobs/coinmarketcap_update.php > /dev/null

Cronjob - New Cryptocurrency

wget -O /dev/null http://www.site-url.com/system/cronjobs/coinmarketcap_newcoin.php > /dev/null

Cronjob - Currency Data

wget -O /dev/null http://www.site-url.com/system/cronjobs/coin_alert_control.php > /dev/null

Cronjob - Created Alerts

wget -O /dev/null wget -O /dev/null http://www.site-url.com/system/cronjobs/currencylayer_api.php > /dev/null
